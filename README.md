### Application Tracking Insights (Google Sheets EDA)

### Sales Performance Dashboard (Excel)
*Built and interactive Excel sales dashboard for Q1 - Q2 2005 using Pivot Tables, slices, covering revenue, profit, orders and profit margin across cities, channels, categories and sales reps. Design KPI cards for Total Revenue, Total Profit, Total Orders Profit Margin, and Average Order Value, alongside visulaizations for monthly trends, order by cities, quantity sold by product, revenue by category and order by channel. 
Tools: Excel, Pivot Tables, Slicers, KPI Cards, Data Visualization, Dashboard Design, Reporting.*

### Super Mart Sales Dashboard (Power BI)
*Built an interactive Power BI dashbaord for Super Mart, tracking Total Sales, Profit and Profit Percentage across monthly, daily, product and category breakdowns.Includes slicers for sale type, payment mode, year, and month, with visualization highlighting top product, top category, sales by payment mode and category performance using a treemap.
Tools: Power BI, Data Visualization,KPI Cards, Slicers, Dashboard Design, Reporting.*

### Bam in Gilead Hospital Patient Analysis (MySQL)
*Wrote SQL queries to analyze a hospital pateint dataset, answering business questions such as average patients age, number of patients diagnosed with hypertension, regions with highest patient counts, average cholesterollevels for diabetics patients, most prescribed medication monthly, visit trends and gender distribution. Also performed data updates using UPDATE statements with safe mode update settings.
